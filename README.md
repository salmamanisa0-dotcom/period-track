# period-track
FLO adalah aplikasi web sederhana untuk mencatat dan memantau siklus menstruasi pengguna. Aplikasi ini dibuat menggunakan HTML, CSS, dan JavaScript dengan penyimpanan data berbasis Local Storage, sehingga data tetap tersimpan di browser tanpa database.

✨ Fitur Utama
📅 Mencatat tanggal mulai menstruasi
📊 Menyimpan riwayat siklus menstruasi
💾 Data tersimpan otomatis di browser (Local Storage)
🗑️ Menghapus data catatan
📱 Tampilan sederhana dan responsif
⚡ Perhitungan siklus otomatis (opsional sesuai fitur yang kamu buat)
🛠️ Teknologi yang Digunakan
HTML5
CSS3
JavaScript (Vanilla JS)
Bootstrap (jika digunakan)
Local Storage API
